# Stats
Each stored face uses 2.2KB of storage (excluding face images which can be deleted once model is trained).

* Chin: 17 points
* Left eyebrow: 5 points
* Right eyebrow: 5 points
* Nose bridge: 4 points
* Nose tip: 5 points
* Left eye: 6 points
* Right eye: 6 points
* Top lip: 12  points
* Bottom lip: 12 points
* Left eye: 6 points
